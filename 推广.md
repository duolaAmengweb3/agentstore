一句话说清楚是什么:
给加密用户和 agent 开发者做的 Wirecutter 式选品站 —— 104 个工具每个都有编辑手写的"最适合谁 / 要避什么坑",不是一堆数字堆出来的榜单。

🌱 为什么做

今年加密 AI agent 生态突然爆炸。OKX 出了 MCP、Binance 出了 Skills、Phantom 出了非托管钱包 MCP、Kraken 甚至搞了个 Rust 单二进制 + NDJSON 输出的 AI-native CLI。Smithery 上加密分类几百个,mcp.so 也是平铺列表,都只告诉你有什么,不告诉你该选谁。

自己做 agent 的时候每次选工具都要翻一上午文档比对,体验太烂。既然没人做,就照 Wirecutter 评电器那个路子自己搞一个 —— 读完文档 + 看过代码 + 跑过真实场景再告诉用户:这类场景首选谁、什么时候别用、和同类比差在哪。

🎯 产品能做什么

1)编辑判断是主角,不是装饰

104 个工具都配了我们手写的 2-3 句判断。举个例子 OKX Agent Trade Kit,我们不会写"官方 MCP,覆盖广",我们写的是"CEX 类里功能覆盖最全的一家,10 个模块含 70+ 技术指标。API key 只在本地,内置只读模式 + 模块过滤 + 限速。想用 OKX 做 agent 交易直接选这个,没有替代方案。"

每条都是这个力度。104 个全部手写,不是批量生成。

2)同类 top-3 横向对比(首页核心板块)

CEX / 链上数据 / AI 框架 / 钱包 / 基建,每板块手挑 top-3,每张卡写清两件事:
"最适合" —— 什么场景选它
"避坑" —— 什么时候别用 / 要注意什么

举个例子选 agent 钱包:
Phantom MCP — 非托管最佳,钥匙不出钱包,stamper keys + auto re-auth。坑:2026-02 才出,生态还在追 Coinbase
Coinbase AgentKit — 要量 / 要稳定,npm 44k/月,x402 支付协议兄弟产品。坑:托管模式,适合"替用户花钱"场景,不是"自己保管 key"
Crossmint — 让 agent 刷信用卡,Visa/Mastercard 已对接,有消费限额。坑:合约型 smart wallet,gas 和复杂度比 EOA 高

看完直接就知道该选谁。

3)分板块安装量榜(真实数据,每 6 小时刷新)

Smithery 调用 + npm 月下载 + GitHub stars,按板块分开排。跨类比数字没意义(Claude Desktop vs Polymarket MCP 不在一条赛道),所以每板块独立榜单。你能直接看到 CEX 类、钱包类、框架类各自谁真的在被用。

4)详情页深度 —— 做一次决定需要的信息全在一页

每个工具点进去你会看到:
编辑判断(2-3 句)
README 自动抓取(about + 功能列表 + 模块表格 + 真实示例命令)
项目活跃度(语言 / license / 贡献者 / 开放 issues / 最后 push / 是否归档)
同类排名(在此类按调用量、star 数、综合分各排第几)
与同类中位数对比的文字说明(高 +340% 还是低 -30%,意味着什么)
一键安装(工具在 Smithery 就直链过去,不在就给具体命令)

5)全自动化,数据不会过期

每 6 小时 GitHub Actions cron 自动拉 GitHub stars / npm 月下载 / Smithery 调用量,写回 md frontmatter。每天扫 Smithery / npm / GitHub 有没有新工具,"信任源"(官方出品、知名作者)自动上架,社区候选开 GitHub Issue 给我人工审。3 个月不更新也不会变成死网站。

6)中英双语,内容双版

每条编辑判断、每个场景入口、每个教程都有中英两版,不是机翻。两边语感差很多,是人写的。
